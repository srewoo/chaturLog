üöÄ Quick Start

Prerequisites

  • Python 3.8+
  • Node.js 16+
  • npm/yarn
  • API keys from OpenAI, Anthropic, and/or Google AI (at least one required)

Clone Repository

git clone <repository-url>
cd chaturLog

Backend Setup

cd backend
pip install -r requirements.txt

# Create .env file
cat > .env << EOF
JWT_SECRET_KEY=your-secret-key-here
CORS_ORIGINS=http://localhost:3000
EOF

# Start server
python server.py
Note: API keys are now configured per-user in the Settings page for security and flexibility.

Frontend Setup

cd frontend
yarn install

# Create .env file
echo "REACT_APP_BACKEND_URL=http://localhost:8001" > .env

# Start dev server
yarn start

Open Application

Navigate to: http://localhost:3000

üéâ Success! ChaturLog is now running. Register an account and start analyzing logs!

üêõ Troubleshooting

Common Issues

Issue: Backend fails to start

Solution: Ensure all Python dependencies are installed:

pip install fastapi uvicorn python-multipart pydantic email-validator PyJWT python-dotenv openai anthropic google-generativeai

Issue: Port already in use

Solution: Kill the process using the port:

# Find process
lsof -ti:8001

# Kill process
kill -9 <PID>

Issue: Git connection fails

Solution: Verify your Git token has read permissions and the repository path is correct (e.g., org/repo for GitHub/GitLab).

Issue: Repository mapping dialog doesn't appear

Possible causes:

  • Repository already detected directly
  • Mapping already exists
  • No service name in logs

Solution: Check Settings ‚Üí Repository Mappings to see existing mappings
